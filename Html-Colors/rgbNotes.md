#RGB 


-->RGB refers Red,Green,Blue colors.
-->rgb(red,green,blue)
-->Each parameter has max value of 255
-->If first parameter is set as 255 and others 0 then it is red colour
-->If 2nd parameter is kept 255 and others 0 then green and similarly 3rd parameter is blue at 255.
-->For white color set all parameters to 255.
-->for black set all parameters to 0.


#Gray color
-->Any shade of gray can be set by keeping the same values for each parameter.


#RGBA
-->Rgba stands for red green blue alpha (it defines opacity of the color)
-->The value of alpha parameter is between 0.0(transparent) to 1.0(opaque).


#HEX colors
-->Hexadecimal colors are specified with #RRGGBB (red,green,blue respectively).
-->Each parameter has max value ff
-->black: #000000
-->White: #ffffff
-->Red: #ff0000
-->Green: #00ff00
-->Blue: #0000ff
-->Gray color is obtained by using same values for all parameters #bebebe


#HSL
-->It defines hue ,saturation and lightness
-->Hue is a degree on color wheel
-->Saturation & Lightness are percentage value
-->For hue 0 is red, 120 is green & 240 is blue
-->0% in saturation means a shade of gray and 100% is the exact color
-->0% in lightness means black and 100% is white.
-->Gray is obtained by keeping the first two parameters  to 0.


#HSLA
-->it extend HSL and contains alpha channel.
-->Alpha parameter's value should be between 0.0 to 1.0
