#Favicon 
   - Favicon is a small image displayed in the browser next to page title.
   - Favicon can be displayed using the link tag.
   - It is written inside the head section.
   - Favicon can be of any format:ico,png,svg.

**Example for displaying favicon

<link rel="icon" type="image/x-icon" href="/revision/favicon.ico">
Here,
    rel="icon" -rel means relationship it tells browser that the file been linked is icon of the webpage.

    type="image/x-icon" -The first part tells us  the category of file (here image) and second part tells us the type of file.

    href="/revision/favicon.ico" -It gives location of the icon file
